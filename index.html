XiaomiTool.

XiaomiTool is an OpenSource Toolkit that has the goal to provide a safe, easy and user-friendly interface to manage your Xiaomi Device.

Supported Devices

  • Xiaomi Mi 2 [aries]
  • Xiaomi Mi 2S [aries]
  • Xiaomi Mi 3W [cancro]
  • Xiaomi Mi 4W [cancro]
  • Xiaomi RedMi 1S [armani]

Features

  • Backup and Restore the device
  • Push files
  • Import camera photos
  • Change RunTime
  • Record Screen
  • Shell on device
  • Install Apps
  • Root (both MiUi stable and Custom roms)
  • Install a custom Recovery
  • Flash Zip files
  • Install Roms
  • Reboot
  • Fork Bomb
